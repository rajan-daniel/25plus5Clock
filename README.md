## ⏱️ Pomodoro Clock (React, HTML, CSS & JavaScript Project)

### 📌 Preview  
A fully functional Pomodoro timer web application built with React that helps users manage focused work sessions and breaks. It includes session control, break control, audio alerts, and a responsive glassmorphism-style UI.

---

### 📖 About the Project  
The **Pomodoro Clock** is a productivity timer application built using React that follows the Pomodoro technique, allowing users to switch between focused work sessions and break periods.

The application includes customizable session and break durations, along with controls to start, pause, and reset the timer. It also automatically switches between session and break modes when the timer reaches zero, accompanied by an audio alert.

A key feature of this project is the use of **React Hooks such as `useState` and `useRef`** to manage timer state, intervals, and mode switching. Instead of relying solely on state re-renders, refs are used to track runtime behavior like whether the timer is running or currently on break.

The UI was designed with a **glassmorphism-inspired style**, using blur effects, transparency, and a background image to create a modern and visually appealing interface.

The project is also **hosted on GitHub Pages**, allowing it to be accessed and used live in the browser.
